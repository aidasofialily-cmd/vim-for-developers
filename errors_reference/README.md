# Developer's Errors and Troubleshooting Reference

Welcome to the **Errors Reference** index. This folder contains structured documentation, guidelines, and troubleshooting tips for common development errors encountered while using standard Vim configurations, Python linters, and frontend layout rendering.

## Table of Contents

- [Vim Common Errors & Pitfalls](./vim_common_errors.md)
  - Details general Vim mistakes, buffer issues, cursor navigation locks, mapping recursion, and parsing traps.
- [Python Pylint Errors](./python_pylint_errors.md)
  - Focuses on Pylint warnings, conventions, code style standards, and resolving errors to meet strict CI pipeline requirements.
- [Frontend Rendering & UI Debugging](./frontend_lint_errors.md)
  - Covers HTML/CSS layout issues, relative path resolutions, CORS barriers, and viewport simulation anomalies within the Website Emulator.

## Purpose & Contribution

Having a dedicated reference guide ensures that developers on Windows 10 targeting Vim environments can quickly query known errors without breaking their active editing loop. When adding new tools or updating configuration mappings, please document any unexpected exceptions or edge-case diagnostics in their corresponding markdown file.
